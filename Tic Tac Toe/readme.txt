********* Assignment3 : Implementation of a tic-tac-toe game using Adversarial Search. ****************

The file tictac.py contains python code that a Tic-Tac-Toe game using Adversarial search (minimax algorithm).

------Dependencies to be used to run the code------------

Python	   : 3.10.12

------How to install the dependencies-----------

Install python 	: sudo apt install python3.10=3.10.12-1

---------How to run the code-------------------------
move to the directory containing "tictac.py" file.
Open terminal and run the command tictac.py 
A GUI board enables the user to play with the AI Player. 
User always starts first.
