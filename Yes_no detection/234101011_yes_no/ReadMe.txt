Step 1. Record the ambient noise i.e. record without speaking anything for few seconds.
Step 2. Save this as ASCII txt file (without header format)
Step 3. Name this file as "Noise.txt". 
Step 4. Now, record Yes/No and save as ASCII txt file (without header format)
Step 5. In line 8 of the program(234101011_yes_no.cpp), mention this filename. 
	By  default the filename is set as "YES.txt"
Step 6. Run the program to see result.

Note: All the recorded files must be copied to "234101011_yes_no\234101011_yes_no" folder
